Abstract

A concise abstract block for the camera-ready paper.

Robots operating with people must interpret language together with embodied cues such as attention, motion, object state, and task history. EDITH studies this problem through an interaction system that connects multimodal perception with task-aware dialogue and adaptive robot behavior.

Method

System components

01

Grounded Perception

Integrates scene state, user attention, and temporal interaction context into a compact representation for decision making.

02

Task-Aware Dialogue

Uses conversational feedback to clarify intent, recover from ambiguity, and keep the robot aligned with the user's goal.

03

Interactive Policy

Selects robot actions that account for task progress, uncertainty, and the cost of asking for more information.
